# Mohd Ayaan — Personal Portfolio

A modern, responsive personal portfolio website built to showcase my profile, skills, projects, education, resume, and contact details in one clean single-page experience.

This portfolio is designed with a strong focus on:
- clean visual hierarchy
- smooth user interactions
- responsive layout across mobile and desktop
- accessibility-friendly structure
- polished front-end presentation
